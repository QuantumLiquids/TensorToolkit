# TensorToolkit User Documentation (EN)

This is the GitHub-first user documentation for QuantumLiquids/TensorToolkit.

## Overview

TensorToolkit is a high-performance, header-only C++ library for tensor-network building blocks.
It provides tensor types, symmetry-aware indices, contractions, decompositions, and MPI/GPU backends
that are used in algorithms such as DMRG, PEPS, TRG, and related workflows.
