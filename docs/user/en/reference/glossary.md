# Glossary

- **Block**: A symmetry-allowed sub-tensor defined by a specific combination of sectors.
- **Block-sparse tensor**: A tensor that stores only symmetry-allowed blocks.
- **Divergence**: The signed sum of quantum numbers across a tensor's indices.
- **Fermionic tensor**: A tensor whose QN type carries fermion parity, enabling sign tracking.
- **Index**: A list of quantum-number sectors plus a direction.
- **IN / OUT**: Direction markers that define how QNs contribute to divergence.
- **QN (Quantum number)**: A symmetry label such as U(1) charge or Z2 parity.
- **Sector**: A pair of (QN, degeneracy) within an index.
- **SVD**: Singular Value Decomposition used for truncation and factorization.
- **TRG**: Tensor Renormalization Group, a coarse-graining algorithm.
