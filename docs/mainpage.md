# TensorToolkit Documentation

TensorToolkit is a high-performance, header-only C++ library for tensor-network
building blocks. It provides symmetry-aware tensors, contractions, and
decompositions, with MPI and GPU extensions.

## Key features

- Symmetry-aware block-sparse tensors
- Fermionic tensor support via Z2 grading
- MPI-enabled decompositions
- Optional GPU acceleration
